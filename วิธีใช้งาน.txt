========================================================================
   คู่มือการติดตั้งและใช้งานโปรแกรมตารางนำขบวน (Motorcade Schedule Trial Version)
========================================================================

โปรแกรมนี้เป็นแบบ Server-based Application เพื่อความปลอดภัยในการเข้ารหัสและเก็บประวัติ 
การย้ายไปรันที่เครื่องอื่นๆ จำเป็นต้องมีสภาพแวดล้อมดังนี้:

------------------------------------------------------------------------
1. ข้อกำหนดของเครื่องคอมพิวเตอร์ที่ใช้งาน (System Requirements)
------------------------------------------------------------------------
- ต้องมีตัวโปรแกรม Python 3 ติดตั้งอยู่ในเครื่อง 
  (หากยังไม่มี สามารถดาวน์โหลดตัวติดตั้งฟรีได้ที่: https://www.python.org/downloads/ )
  *ข้อแนะนำตอนติดตั้ง:* ตอนเปิดตัวติดตั้ง ให้ติ๊กเลือกช่อง "Add Python.exe to PATH" เสมอ

------------------------------------------------------------------------
2. วิธีการรันโปรแกรมเพื่อใช้งาน (ง่ายที่สุด)
------------------------------------------------------------------------
- แตกไฟล์ ZIP "Movetabel V1.zip" ออกมาเป็นโฟลเดอร์ปกติ
- ดับเบิ้ลคลิกไฟล์ "รันโปรแกรม.bat" 
- ตัวไฟล์ Bat จะทำการตรวจสอบระบบ หากไม่มีไลบรารีที่จำเป็น (เช่น Flask) จะทำการดาวน์โหลดและติดตั้งให้โดยอัตโนมัติ (ในขั้นตอนติดตั้งนี้จำเป็นต้องมีการเชื่อมต่ออินเทอร์เน็ต)
- จากนั้นบราวเซอร์จะเปิดหน้าโปรแกรมตารางนำขบวน (`index.html`) ขึ้นมาให้ใช้งานทันที

*ข้อสำคัญ:* โปรดเปิดหน้าต่างดำๆ (Command Prompt) ของไฟล์ Bat ทิ้งไว้ระหว่างใช้งานโปรแกรม และเมื่อใช้งานเสร็จแล้ว สามารถปิดหน้าต่างนั้นเพื่อหยุดการทำงานได้เลยครับ

------------------------------------------------------------------------
3. ข้อมูลบัญชีสำหรับทดลองใช้งาน
------------------------------------------------------------------------
เนื่องจากฐานข้อมูลเดิมถูกคัดลอกติดไปด้วย คุณสามารถเข้าใช้งานได้ด้วยสิทธิ์เดิม:
- ชื่อผู้ใช้: nattapon
- รหัสผ่าน: รหัสผ่านของคุณ (เนื่องจากถูกรีเซ็ต คุณสามารถพิมพ์รหัสผ่านใหม่ใดก็ได้ในครั้งแรก ระบบจะบันทึกรหัสผ่านใหม่ให้ทันทีครับ)

------------------------------------------------------------------------
4. วิธีการนำขึ้นอินเทอร์เน็ตใช้งานฟรี (Render & Neon Cloud Hosting)
------------------------------------------------------------------------
หากต้องการให้ทุกคนสามารถใช้งานโปรแกรมนี้พร้อมกันแบบเรียลไทม์ผ่านอินเทอร์เน็ตได้ฟรี ให้ดำเนินการดังนี้ครับ:

ขั้นตอนที่ 4.1: สมัครใช้งานฐานข้อมูลคลาวด์ฟรี (Neon.tech)
1. เข้าเว็บไซต์ https://neon.tech และสมัครบัญชีฟรี (ไม่ต้องใช้บัตรเครดิต)
2. สร้างโปรเจกต์ใหม่และคัดลอก "Connection String" (เช่น postgresql://[user]:[password]@[host]/[dbname]) เก็บไว้
3. ฐานข้อมูลนี้จะทำหน้าที่เก็บข้อมูลตารางนำขบวนและบัญชีผู้ใช้ออนไลน์ของคุณอย่างถาวร

ขั้นตอนที่ 4.2: อัปโหลดโค้ดขึ้น GitHub และ Render.com
1. สมัครใช้งาน GitHub (https://github.com) และสร้าง Repository ใหม่
2. อัปโหลดไฟล์ในชุดนี้ (ได้แก่ app.py, index.html, requirements.txt) ขึ้นไปบน GitHub
3. สมัครบัญชีฟรีที่ https://render.com
4. เชื่อมต่อบัญชี GitHub ของคุณกับ Render:
   - สร้าง "Web Service" (สำหรับเซิร์ฟเวอร์หลังบ้าน Python Flask):
     * เลือก Repository โค้ดโปรแกรมของคุณ
     * เลือก Runtime: Python 3
     * กำหนด Build Command: pip install -r requirements.txt
     * กำหนด Start Command: gunicorn app:app
     * เข้าไปที่แท็บ Environment ในหน้าเว็บ Render แล้วเพิ่มตัวแปร (Environment Variable) ดังนี้:
       * ชื่อตัวแปร: DATABASE_URL -> ค่าของตัวแปร: [วาง Connection String ที่ได้มาจาก Neon.tech ในข้อ 4.1]
   - สร้าง "Static Site" (สำหรับหน้าจอหลักหน้าบ้าน index.html):
     * เลือก Repository เดิม หรือนำ index.html ไปฝากเว็บฟรี เช่น GitHub Pages / Vercel
5. เมื่อ Render ทำการ Deploy สำเร็จ คุณจะได้ URL ของหลังบ้าน (เช่น https://my-app.onrender.com)
6. ให้นำ URL ที่ได้นี้ ไปแก้ไขค่า const API_URL = 'http://localhost:5000/api'; ในบรรทัดที่ 613 ของไฟล์ index.html ให้ชี้ไปยัง URL ใหม่ของคุณ (เช่น const API_URL = 'https://my-app.onrender.com/api';) แล้วบันทึกไฟล์ส่งขึ้น GitHub อีกครั้ง

ระบบของคุณก็จะออนไลน์ใช้งานฟรีพร้อมระบบความปลอดภัยเต็มรูปแบบบนอินเทอร์เน็ตทันทีครับ!
